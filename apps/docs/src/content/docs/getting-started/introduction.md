# Minions Evaluations

Benchmarks, test cases, and prompt scoring for continuous agent improvement

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-evaluations/sdk`) — Core library for Node.js
- **Python SDK** (`minions-evaluations`) — Python bindings
- **CLI Tool** (`@minions-evaluations/cli`) — Command-line interface
- **Web App** — Interactive playground at [evaluations.minions.wtf](https://evaluations.minions.wtf)
