# Welcome to Minions Evaluations

We're excited to introduce **Minions Evaluations** — Benchmarks, test cases, and prompt scoring for continuous agent improvement.

## What's Included

- **TypeScript SDK**: `npm install @minions-evaluations/sdk`
- **Python SDK**: `pip install minions-evaluations`
- **CLI**: `npm install -g @minions-evaluations/cli`

## Getting Started

Check out the [documentation](https://evaluations.minions.help) for a full guide.
